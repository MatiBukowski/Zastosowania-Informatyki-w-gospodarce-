As for data management there will be two scenarios: data prediction and data analysis.

## Data Analysis

When it comes to data analysis I think the first and basic functionality will be correct display of information to the end user which in this scenario will be worker of the restaurant on web frontend view of this application.

We should first settle on basic scenarios for data display that shall be dependent on different time scopes (per each hour of a day, per entire week etc.) like:

- Amount of people that attended the restaurant
- Amount of certain menu item sold
- Attractiveness of a table (which table was picked the most)
- Different type of reservations

From current scope (16.03.2026) writing other requirements is too extreme as I cannot predict what different data analysis we might want so we shall implement those that are above and go from there.
